# ⚡ TaskMaster — Focus Mode To-Do

[🔗 Живое демо (Vercel) — Скоро здесь будет ссылка]

Современное SPA-приложение для управления задачами. Фокус на чистоте кода, плавных анимациях и сохранении данных.

## ✨ Особенности
- **Persistent State:** Все задачи сохраняются в `localStorage`.
- **Framer Motion:** Плавные переходы и анимация удаления (layout animations).
- **Smart Filters:** Фильтрация по статусу (Все / В работе / Готово).
- **Dark Tech UI:** Глубокий черный фон и неоновые акценты.

## 🛠 Стек
- **React 18** (Vite)
- **Framer Motion** (Анимации)
- **CSS Modules**
- **Lucide React** (Иконки, если добавим)
