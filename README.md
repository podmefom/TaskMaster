# ⚡ TASKMASTER [CORE_OS v2.0]

[🔗 Живое демо (Vercel)](https://task-master-five-tau.vercel.app/)

Интерактивное SPA-приложение, созданное на стыке продуктивности и агрессивного Dark Tech дизайна. Это не просто список задач, а интерфейс управления процессами с глубокой визуальной и аудио-обратной связью.

## 🛠 Технологический стек
- **React 19** — использование новейших хуков и оптимизаций.
- **Framer Motion** — сложные layout-анимации для бесшовного UX.
- **Web Audio API** — кастомный звуковой движок на базе осцилляторов для тактильного отклика системы.
- **Lucide React** — современный сет системных иконок.

## ⚡ Ключевые особенности
- **Аудио-отклик:** Каждое действие (добавление, выполнение, удаление) озвучено уникальным цифровым паттерном.
- **Engine Status:** Динамический расчет прогресса выполнения задач, интегрированный в заголовок (как статус сборки двигателя).
- **CRT & Glitch Visuals:** Эффект сканирующих линий, мерцания и "стеклянного" интерфейса (Backdrop Blur).
- **Zero-Data Loss:** Полная синхронизация состояния с `localStorage`.
